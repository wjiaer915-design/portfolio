01 / BRAND IDENTITY

躺平 TANGPING

以“拒绝内卷，回归自在”为核心理念的品牌全案设计。通过低饱和度的色彩体系与极简的视觉符号，传达松弛感与治愈力，构建与 Z 世代产生精神共鸣的栖息地。

02 / FESTIVAL IP & VISUAL

NOISE FEST 音乐节
视觉识别系统

将声波的无序律动转化为视觉秩序。负责音乐节全套 IP 设定及主视觉延展。利用高对比度的酸性设计风格（Acid Graphics）与 3D 动效，构建极具冲击力的现场视觉体验与周边衍生品体系。

  • Key Visual
  • Stage Design
  • Merch
03 / UI & UX DESIGN

App 界面设计
